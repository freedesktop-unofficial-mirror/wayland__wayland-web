This page describes a few other things that work with Wayland that are not included in the Wayland or Weston repos.

mplayer2 port by Alexander Preisinger 

    git clone https://github.com/giselher/mplayer2-wayland.git
    cd mplayer2-wayland
    ./configure --prefix=$prefix --enable-wayland --enable-debug --disable-x11
    make
    make install
